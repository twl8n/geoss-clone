Instructions:
Click the pencil to select a node. Once selected, the node will be gold colored.

To modify a node's properties, select the node and modify the properties on the right hand side. Changes can be saved by pressing 'Update', or running the tree, or selecting a different node.

To add a node, select the parent node, then choose the type of analysis from the drop down menu at the bottom of the page. Click Add Node.

To delete a node, select the node, then click the trash can.

To run the tree, click 'Run Analysis'. All nodes from the selected node down will be run.
