Description
The permCluster analysis performs hierarchical clustering analysis for microarray data (it is recommended to apply this module for an array data set with less than 2,000 genes for optimal performance). One can choose different combinations of one of the distance metrics (e.g., Euclidean, Pearson correlation [default], Spearman correlation) and one of clustering algorithms (e.g., "single", "complete", "average" [default]).
See also the help pages at Rcluster help


Input files
PermCluster expects input in this form:

  • a file containing an analysis criteria. The file should have row and column labels. Note that we won't process more than 7000 rows.

Analyses that create input in this format include:
  • filter
  • diffDiscover (likely too many rows - run through filter first)

 

Output files
PermCluster creates the following output:

  • A file containing data necessary to draw a graph
  • A file containing user data
  • A file containing stat data
  • A file linking the graph file and a data file (needed for input to treedraw)
Analyses that accept permCluster output as input include:
  • treedraw

 

Parameters:

  • Need of log 2 transformation (T, F) - The filter analysis provides log transformed data, so this should be set to 'F' if data is received from filter.
  • Euclidean distance measure (euc, cor, spearman) - the distance/dissimilarity metric. One of the following:
    "euclidean" - standard euclidean distance
    "cor" - the Pearson correlation coefficient
    "spearman" - the Spearman correlation coefficient
