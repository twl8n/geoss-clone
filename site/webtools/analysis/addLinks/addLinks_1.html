Description
The addLinks module allows users to add html links to their output. The links take the user to the appropriate information for each gene at the NCBI site. This linking is similar to the links created by the treedraw analysis, but addLinks can be applied to the output of many modules. Options all the links to be placed at the front or back of the output file. The output file can be in either html or txt format.


Input files
Add links expects input in this form:

  • a file containing a column labeled "Probe.Set" that contains the gene accession number.

Analyses that create input in this format include:
  • qualityControl
  • diffDiscover
  • filter

 

Output files
AddLinks creates the following output:

  • A html or txt file that contains the columns of the input file, plus a column of linking gene accession numbers to the associate NCBI information.
Analyses that accept AddLinks output as input include:
  • none

 

Parameters:

  • Log filename - name of the file containing log and error messages
  • Output filename - name of the file containing new links
  • Output type - specifies whether the output file should be html or txt format
  • Add NCBI link - currently this is the only type of link to add. If this option is not checked, your output file would look exactly like your input file. Eventually, we hope to add links to other sites.
  • Placement of links - whether to place the new information as the first column of the file or as the last
