Description
The GEOSS module 'diffDiscover' performs calculations that are intended for discovery of differentially expressed genes. It computes the following:

  • Hypothesis testing by LPE method.
  • Fold change in the log scale.
  • Hypothesis testing by t-test.
    The output from this module is formatted in a table structure for easier access and readability.
    Also includes procedures for controlling the false discovery rate (FDR): Benjamini & Hochberg (1995), Benjamini & Yekutieli (2001) step-up procedures.


    Statistical Methods
    The following computations are done on the input data:

  • Traditional fold changes are computed by a log-base-2 transformation on the ratio of average expression values between two conditions.
  • Log fold changes are computed as the difference between average expression values of the two conditions, where the expression values have been log-base-2 transformed before averaging.
  • The student's t-test is performed on the log-base-2 transformed expression values, for all pairs of conditions.
  • Local pooled error (LPE) test is performed on the log-base-2 transformed expression values, for all pairs of conditions. For explanation of the LPE test, look at:

    1. Nitin Jain, Jayant Thatte, Thomas Braciale, Klaus Ley, Michael O'Connell, and Jae K. Lee (2003). Local pooled error test for identifying differentially expressed genes with a small number of replicated microarrays, to appear in Bioinformatics.

    2. Lee, J.K. and O'Connell, M. (2003). Chapter 7: An Splus library for the analysis of microarray data. The Analysis of Gene Expression Data: Methods and Software, edited by Parmigiani, Garrett, Irizarry, and Zeger. Springer: New York.


    Input files
    Differential Discovery expects input in this form:

    • a file containing chip data. The file should have row and column labels.

    Analyses that create input in this format include:
    • qualityControl

     

    Output files
    Differential Discovery creates the following output:

    • An intermediate data file containing fold change, LPE, and T-test values.

    Analyses that accept differential discovery output as input include:
    • filter
    • permCluster (This runs but is not recommended. Data should be filtered first.)

     

    Parameters:

    • Intermediate data filename - name of the data file to pass to other analysis nodes (contains LPE, T-test, and fold change values)
    • Log filename - name of the file containing log and error messages
