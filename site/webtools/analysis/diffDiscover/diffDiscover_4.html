Description
The GEOSS module 'Differential Discovery' performs calculations that are intended for discovery of differentially expressed genes. This module uses the following methods: fold changes, two-sample t-test, Westfall & Young's permutation test, LPE test, and SAM test. The output from this module is formatted in a table structure for easier access and readability.
Differential Discovery version 3 uses R 2.1. Earlier nodes use R 1.9.


Statistical Methods
The following computations are done on the input data::

  • Traditional fold changes (TradFC) are computed by a log-base-2 transformation on the ratio of average expression values between two conditions.
  • Log fold changes (LogFC) are computed as the difference between average expression values of the two conditions, where the expression values have been log-base-2 transformed before averaging.
  • Student's two-sample t-test is the standard statistical procedure for comparing two conditions.
  • Westfall & Young's permutation test is a resampling-bsed procedure for comparing two conditions. For explanation of the permutation test analysis, look at:
      1. Westfall, P.H., and Young, S.S. (1993). Resampling-based multiple testing: Examples and methods for p-value adjustment. John Wiley & Sons..
      2. Ge, Y., Dudoit, S., and T. P. Speed (2003). Resampling-based multiple testing for microarray data analysis. TEST, Vol. 12, No. 1, p. 1-44.
  • Local Pooled Error (LPE) test is performed on the log-base-2 transformed expression values, for all pairs of conditions. For explanation of the LPE test, look at:
      1. Jain, N., Thatte, J., Braciale, T., Ley, K., O'Connell, M., and Lee, J.K. (2003). Local pooled error test for identifying differentially expressed genes with a small number of replicated microarrays, Bioinformatics, 19, 1945-1951.
      2. Lee, J.K. and O'Connell, M. (2003). Chapter 7: An Splus library for the analysis of microarray data. The Analysis of Gene Expression Data: Methods and Software, edited by Parmigiani, Garrett, Irizarry, and Zeger. Springer: New York.
  • Significance Analysis of Microarray (SAM) is performed on the log-base-2 transformed expression values, for all pairs of conditions. For explanation of the SAM analysis, look at:
      1. Tusher, V.G., Tibshirani, R., and Chu, G. (2001). Significance analysis of microarrays applied to the ionizing radiation response, PNAS , 98, 5116-5121.
      2. Storey, J.D. (2002). A direct approach to the false discovery rate, _Journal of the Royal Statistical Society, Series B, 64, 479-498.
      3. Storey, J.D. and Tibshirani, R.(2003). Chapter 12: SAM Thresholding and False Discovery Rates for Detecting Differential Gene Expression in DNA Microarrays The Analysis of Gene Expression Data: Methods and Software, edited by Parmigiani, Garrett, Irizarry, and Zeger. Springer: New York


    Input files
    Differential Discovery expects input in this form:

    • A file containing chip data. The file should have row and column labels.
    Analyses that create input in this format include:
    • Quality Control

     

    Output files
    Differential Discovery creates the following output:

    • An intermediate data file containing expression data and statistics for all probe sets.
    Analyses that accept differential discovery output as input include:
    • Filter
    • Add Links

     

    Parameters:

    • (Optional) differential discovery by - computationally expensive LPE FDR and SAM q-value are calculated

     

    Note:

    • If there are no replicates for a condition, only the fold changes are computed.
