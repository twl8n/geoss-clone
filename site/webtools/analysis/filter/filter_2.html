Description
The module 'Filter' allows one to triage genes for further analysis, such as clustering and functional analysis, based on a combination of several statistical criteria. In addition, genes related to some key words can be selected based on gene symbol, gene name, and gene ontology (GO) databases. This module also allows one to add annotations such gene symbol, gene names, unigene ID, locus ID, chromosome location, and GO information. The order of conditions can be changed in this module.


Input files
Filter expects input in this form:

  • A file containing differential discovery results. The file should have row and column labels.

Analyses that create input in this format include:
  • Differential Discovery

 

Output files
Filter creates the following output:

  • A filtered intermediate data file containing a subset of input rows
  • A filtered output file containing a subset of input rows and statistics
Analyses that accept filter output as input include:
  • Cluster
  • Add Links

 

Parameters:

  • Two conditions to compare: vector levels of two conditions for comparison
  • First Filtering: choice of statistic for primary filtering
  • Second Filtering: choice of statistic for additional filtering
  • Cutoff values: cutoff values for filtering by the selected statistic
  • Functional Filtering: key words for gene symbol, gene name, or GO-based filtering
  • Maximum number of rows to return: maximum number of genes
  • Condition order: re-order conditions
  • Output data filename: name of the data file with all data and statistics
  • Intermediate data filename: name of the data file to pass to other analysis nodes
  • Log filename: name of the file containing log and error messages

 

Note:

  • If no statistics are selected in 'Filter', the LPE z-statistic will be used to select the maximum number of genes.
  • If a statistic that was not computed in 'Differential Discovery is selected in 'Filter', the LPE z-statistic will be used to select the maximum number of genes.
  • If there are no replicates for a condition, the LogFC will be used to select the maximum number of genes.
  • If 1900 genes are kept, the files generated with 'Cluster' will not be viewable with Acrobat Reader.
