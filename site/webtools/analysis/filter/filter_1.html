Description
The filter module allows one to triage genes for further analysis, such as clustering and functional analysis, based on a combination of several criteria, such as fold change, two-sample t-test p-value, and LPE-test p-value.
Reference: Nitin Jain, Jayant Thatte, Thomas Braciale, Klaus Ley, Michael O'Connell, and Jae K. Lee (2003). Local pooled error test for identifying differentially expressed genes with a small number of replicated microarrays, to appear in Bioinformatics.


Input files
Filter expects input in this form:

  • a file containing differential discovery results. The file should have row and column labels.

Analyses that create input in this format include:
  • diffDiscover

 

Output files
Filter creates the following output:

  • A filtered data file containing a subset of input rows
Analyses that accept filter output as input include:
  • permCluster

 

Parameters:

  • Two conditions to compare - vector levels of two conditions for comparison
  • Filter on t-p-value - whether to use t-statistics for filtering
  • Select for t-p-value <= - t-stat p-value
  • Maximum number of rows to return - maximum number of genes by t-stat (if you exceed 1900 rows, the files generated with the treedraw module will not be viewable with Acrobat Reader)
  • Filter on LPE - whether to use LPE test for filtering
  • Select for LPE values <= - LPE FDR p-value (Benjamini & Yekutieli, 2001)
  • Maximum number of rows to return - maximum number of genes by LPE
  • Filter on LPEBY - whether to use LPE test for filtering
  • Select for LPEBY values <= - LPEBY FDR p-value (Benjamini & Yekutieli, 2001)
  • Maximum number of rows to return - maximum number of genes by LPEBY
  • Filter on fold change - whether to use fold change criterion for testing
  • Select for fold change values >= - fold change value
  • Maximum number of rows to return - maximum number of genes by FC
  • Intermediate data filename - name of the data file to pass to other analysis nodes
  • Log filename - name of the file containing log and error messages


Note: If no criteria are selected for filtering, LPE will be used. If multiple criteria are selected, the order of precedence is: LPE, LPEBY, TP, FC.
