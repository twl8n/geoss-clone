Description
The GEOSS module 'Multi-condition Differential Discovery' performs calculations that are intended for discovery of differentially expressed genes under multiple conditions. This module compares multiple conditions simultaneously while 'Differential Discovery' compares all possible pairs of multiple conditions. This module uses the following methods: ANOVA and HEM. The output from this module is formatted in a table structure for easier access and readability. Multi-condition Differential Discovery version 1 uses R 2.1.


Statistical Methods
The following computations are done on the input data::

  • ANOVA (Analysis of Variance) is a standard statistical procedure for comparing multiple conditions.

  • WY (Westfall & Young) and BH (Benjamini & Hochberg) tests compute adjusted p-values for multiple testing procedures. For explanation of the WY and BH tests, look at:
      1. Westfall, P.H., and Young, S.S. (1993). Resampling-based multiple testing: examples and methods for p-value adjustment. John Wiley & Sons.
      2. Benjamini, Y. and Hochberg, Y. (1995). Controlling the false discovery rate: a practical and powerful approach to multiple testing. J. R. Statist. Soc. B. 57: 289-300.
      3. Ge, Y., Dudoit, S., and T. P. Speed (2003). Resampling-based multiple testing for microarray data analysis. Test. 12 (1): 1-44.
  • HEM (Heterogeneous Error Model) is a statistically powerful procedure for multiple conditions, particularly, with a small number of replicates. For explanation of the HEM test, look at:
      1. Cho, H. and Lee, J.K. (2004). Bayesian Hierarchical Error Model for Analysis of Gene Expression Data, Bioinformatics, 20, 2016-2025.
