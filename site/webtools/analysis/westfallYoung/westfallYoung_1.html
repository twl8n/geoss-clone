Description
The westfallYoung analysis provides permutation-based multiple testing procedures to identify differentially expressed genes in DNA microarray experiments. It includes resampling-based multiple testing procedures for controlling the family-wise error rate (FWER): Bonferroni, Hochberg (1988), Holm (1979), Sidak, Westfall & Young (1993) (referred to as minP and maxT).
WestfallYoung also includes procedures for controlling the false discovery rate (FDR): Benjamini & Hochberg (1995), Benjamini & Yekutieli (2001) step-up procedures.


Input files
WestfallYoung expects input in this form:

  • a file containing quality controlled chip data. The file should have row and column labels.

Analyses that create input in this format include:
  • QualityControl

 

Output files
WestfallYoung creates the following output:

  • An intermediate data file
Analyses that accept WestfallYoung output as input include:
  • none

 

Parameters:

  • Intermediate data filename - name of the data file to pass to other analysis nodes
  • Log filename - name of the file containing log and error messages
