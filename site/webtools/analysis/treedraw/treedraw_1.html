Description
The treedraw analysis draws the hierarchical clustering tree and corresponding image maps (cluster-image map; CIM) from the output of a hierarchical clustering algorithm. This will generate a high-density pdf graphic, which is zoomable by utilizing Adobe Acrobat reader. The gene names and annotations at the end of the tree branches are hyperlinked to genome web sites.


Input files
Treedraw expects input in this form:

  • a file containing the names of a graph file and a data file.

Analyses that create input in this format include:
  • permCluster

 

Output files
Treedraw creates the following output:

  • A pdf file showing a cluster tree
Analyses that accept treedraw output as input include:
  • none

 

Parameters:

  • Log filename - name of the file containing log and error messages
  • Graphical output filename - name of the file containing the tree
  • Zoom factor - zoom increases "magnification" of the tree but not the colorized expression map
