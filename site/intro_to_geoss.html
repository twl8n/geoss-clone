GEOSS: Microarray Analysis and Archive
Created at the University of Virginia

What is GEOSS?
GEOSS is a repository and data analysis suite for gene microarray data. The acronym stands for Gene Expression Open Source System.

Researchers at the University of Virginia
If you are performing microarray studies with the BRF at UVA, you'll need an account on the UVA GEOSS system. Please email geoss-help@virginia.edu for an account. To see learn about using GEOSS, please see our Step By Step Guide.

Downloading GEOSS
The GEOSS package is also available as a download. With its easy to use web interface, strong security and ever expanding analysis capabilities, GEOSS is an ideal way to manage microarray data. It also includes features to help your microarray center manage the chip workflow. GEOSS is a mature, open source package freely available. Running GEOSS requires a Linux server. Installation takes several hours. Extensive documentation is available for administrators and end users. The GEOSS developers are available for questions via email. For more details about downloading, see the download page.

From the Researcher's Point of View
We built GEOSS to support genome-wide microarray studies at the University of Virginia. Base features are data repository, database, Analysis Tree analysis, security, and microarray center workflow and chip tracking.

From the investigator's point of view, GEOSS is the place to create orders for genechip microarray assays, retrieve data for completed assays, and analyze data. If the time comes when the data needs to be made publicly accessible for purposes of publication, GEOSS also has a MIAME complaint public data repository.

GMB array data analysis service
In addition to the many features of the GEOSS database and analysis system, the GMB Core offers a formal analysis service to University of Virginia array users. This analysis service will provide the results of quality control, statistical evaluation and discovery of differential expression, clustering & classification analysis, and some annotation analysis for each array study. This service is partially supported by the UVA School of Medicine, but some fees will be assessed starting in January 1, 2003. The BRF will continue provide basic analysis of each array data set.
