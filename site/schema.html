Schema Description Images

This page has 5 large images depicting the database schema of GEOSS. These images are provided as a technical reference for programmers and database administrators. This material is not necessary (nor even helpful) for investigators using GEOSS.

There is a short description above each image.

 

The first image below depicts the tables which hold the conceptual criteria for the investigator's experiments as well as the tables which hold the derived Affymetrix spot values.


 

The following tables are associated with lab orders. They include information regarding the physical sample, hybridization(s) and billing.
